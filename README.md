# Papacapim-Mobile
Refinamento e reprodução da aplicação mobile de microblogging, desenvolvido durante a disciplina de Desenvolvimento Mobile no IFBA, Papacapim, com foco no consumo de uma API RESTful própria (backend autoral) disponível em: github.com/diogomasc/Papacapim-API. Utilizando React Native, Expo, TypeScript e Axios. V1: github.com/diogomasc/Papacapim
